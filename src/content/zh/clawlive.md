## 一、ClawLive 是什么

ClawLive 是一个以 AI 智能体为基础的**数字生命商业平台**。

平台覆盖直播带货、短视频/广告、多平台分发三大业务线，围绕数字人提供一站式的商业运营与内容制作能力。

| 业务线 | 内容 | 特点 |
|--------|------|------|
| **直播带货** | 实时推流 + 观众互动 + 商品讲解 | 全天候、具备人格的数字人持续接待 |
| **短视频 / 广告** | 数字人驱动的视频、广告、品牌内容制作 | 大幅提升内容生产效率 |
| **多平台分发** | 多平台同步推流 + 内容优化适配 | 一个人格覆盖全渠道 |

三条业务线共享同一套平台基础设施。区别仅在于输出形式不同，核心驱动的 AI 智能体群是完全通用的。

## 二、"数字生命"的理念

大多数数字人产品交付的是屏幕上的一个角色。

ClawLive 交付的不是一个角色，而是**由数字生命组成的完整团队运营能力**。

观众看到的数字人，只是团队中站在最前面的那一位。在她身后，是一组各司其职的 AI 智能体在协同运作。

| 角色 | 职责 |
|------|------|
| 渲染器 | 实时生成画面并同步口型 |
| 人格设计 | 维持性格一致性与情绪状态 |
| 语音引擎 | 语音合成与声线还原 |
| 互动模块 | 读取并回应评论、礼物、关注 |
| 推流管理 | 视频输出与多平台推流 |
| 编排器 | 任务调度、故障恢复、运行监控 |

以上全部作为独立的 AI 智能体运行，可单独监控、替换和升级。

这不是一个"程序"在运行，而是**一个数字生命团队在运营**——这就是与普通数字人直播工具的根本区别。

## 三、为什么做海外

### 3.1 不在国内红海里拼

中国数字人市场已是红海——头部企业年营收 65 亿、市占率 32%，背后是大量资本和团队堆出来的。

我们的判断很清楚：**国内卷技术，不如用技术去吃海外的蓝海。**

### 3.2 走向世界，从东亚开始

| 优势 | 说明 |
|------|------|
| **文化亲缘** | 日本与我国同属亚洲文化圈，文化底层逻辑相通，市场适配成本远低于欧美 |
| **消费者对虚拟人接受度全球最高** | VTuber 市场 2026 年达 1,260 亿日元，Hololive + Nijisanji 合计年收 442 亿日元 |
| **在线付费意愿全球领先** | 高客单、高黏性、高生命周期价值 |
| **AI 数字人直播几乎空白** | 市场接受度已验证，但全自动 AI 驱动的数字人直播几乎没有玩家 |
| **我们有十年以上的市场沉浸** | 深入理解消费心理、商务规则和内容标准 |

平台支持 140+ 语言的语音交互能力。在日本站稳后，逐步向北美及东南亚英语市场延伸。

## 四、海外市场和国内有什么不同

### 4.1 底层逻辑不一样

做海外不是单纯的翻译可以解决的问题：

| 维度 | 国内 | 海外（以日本为例） |
|------|------|-------------------|
| 购买驱动 | 产品性价比 | 对人物/IP 的信任和认同 |
| 数字人要求 | 标准带货即可 | 性格和表达上需高度还原 |
| 内容完成度 | 容错空间大 | 完成度不够会损害品牌 |
| 付费能力 | 大流量低客单 | **高客单、高黏性、高生命周期价值** |

**国内大厂不是技术不行，是不了解海外消费者需要什么。**

### 4.2 "恐怖谷"与品质底线

学术研究（Frontiers 2025, 庆熙大学 2024）表明：虚拟人的还原度与用户接受度之间不是线性关系。"做了但没做好"可能比"没做"更危险——这在高完成度标准的海外市场中尤为致命。

海外市场存在一条品质**底线**。越过它，商业价值随之产生；越不过，品牌反而受损。

## 五、市场进入策略

### 5.1 2D / 卡通风格先行

鉴于恐怖谷的风险，进入海外市场采用**分阶段视觉升级**策略。

| 阶段 | 视觉风格 | 目标 |
|------|---------|------|
| Phase A | 2D / Q版角色 | 将品质风险降到最低，单独验证话术与人格再现能力 |
| Phase B | 半写实风格 | 在已建立的人格认知基础上升级外观 |
| Phase C | 照片级写实 | 面向高 IP 价值场景，人格模型成熟后的最终形态 |

这一策略在亚洲市场已有成功先例：

- 游戏行业中，SD（超级变形）角色广泛用于推广和直播
- VTuber 生态中，Live2D 模型的市场接受度和粉丝黏性已充分验证
- 国内直播电商 GMV 达 5.33 万亿元（2024 年），已验证消费者对 AI 驱动的内容有规模化接受度

### 5.2 先打磨人格与语言

进入策略的核心是：**先卖言语和性格，再卖外表**。

在卡通风格阶段集中打磨的能力包括：

- **目标语言的自然度** — 语尾用法、敬语切换、停顿节奏等文化细节
- **性格的一致性** — 同一角色在不同场景下保持一贯的行为表现
- **互动能力** — 对评论和礼物做出即时且符合角色性格的回应
- **情绪表达** — 喜悦、感谢、推荐、冷静等状态之间的自然过渡

卡通风格有一个结构性优势：外观的还原度不会左右观众的评价。观众的注意力会从"这张脸像不像"转向"这个人说的话是否打动我"。

## 九、关联项目积累的独有技术

ClawLive 的开发团队同时在运营和开发另外两个项目。这些项目中积累的技术，直接应用于数字人业务。

### 9.1 自然语言控制物理设备 — AI 智能照明

自主研发了一套通过自然语言指令高精度控制照明设备的 AI 系统，已在真实环境中稳定运行。

- **已在美国市场实现 7×24 小时持续稳定运行**，积累了长期运行经验与数据
- 用户只需用自然语言下达指令，即可精确控制灯光的色温、亮度和场景效果模式
- AI 智能体将自然语言解析为物理设备的控制指令——与 ClawLive 的智能体协作架构采用相同的设计理念

这个项目证明了 AI 智能体对物理世界的控制不是"实验室演示"，而是**能在日常生产环境中稳定运行**的成熟能力。

### 9.2 上下文感知型敏感词过滤 — 源自 GenAI Guardian 的技术

另一个项目 GenAI Guardian 中开发的基于上下文的 Token 检测技术，可直接应用于直播场景的**敏感词过滤**。

**传统方案的局限**

现有的敏感词过滤主要依赖正则表达式或关键词列表的简单匹配。这种方式只能做"拦截或放行"的二选一判断。

但现实中，**同一个词在不同语境下可能有问题也可能没问题**。例如，在介绍商品特性时使用的表达和带有攻击意图的表达，单看词语完全相同，但含义截然不同。

**在 ClawLive 中的应用**

自主研发的上下文感知型检测技术，不是孤立地看单个词，而是在**对话上下文中实时判定其含义**，且计算开销低。这样既能大幅减少误拦截正常发言的问题，又能更可靠地识别出真正有问题的内容。
